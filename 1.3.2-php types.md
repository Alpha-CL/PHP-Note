#### integer

整数可以十进制（基数10），十六进制（基数16），八进制（基数8）或二进制（基数2）负整数 表示

> 要使用八进制符号，请在数字前加上0（零）
> 要使用十六进制表示法，请在数字前加0x
> 要使用二进制表示法，请在数字前加0b


``` php


$a = 1234;                  十进制

$a = 0123                   八进制

$a = $x1A;                  十六进制

$a = 0b11111111             二进制



```
